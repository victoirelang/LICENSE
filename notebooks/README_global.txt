# Projet de Programmation Chimie

## Description
Ce notebook Jupyter démontre l'utilisation de fonctions personnalisées pour filtrer et retirer des éléments d'une base de données. Il utilise Pandas pour charger les données à partir d'un fichier CSV et applique plusieurs fonctions de filtrage pour préparer les données pour une analyse ultérieure.
